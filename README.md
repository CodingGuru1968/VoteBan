# 🚀 Introduction

**VoteBan** gives your regular players a way to deal with rule-breakers when staff aren't around. Instead of just waiting for an admin to show up, players can start a vote to kick, mute, or ban someone. It uses your existing punishment commands, so it will work with any setup you already have.

### ⭐ Key Features

* Let your players help: Keep the server clean by letting your community help out when staff are busy or offline.
* Works with your current setup: It uses your existing ban, kick, and mute commands, so you don't have to change your workflow.
* Stops abuse: Built-in "already-voted" settings and strict permission requirements keep players from spamming votes on each other.
* Flexible voting: You choose if a vote succeeds based on a specific number of players (e.g., 10 votes) or a percentage of the server.
* Keeps everyone informed: Optionally broadcast updates in chat or lock the chat during a vote to keep things focused.
